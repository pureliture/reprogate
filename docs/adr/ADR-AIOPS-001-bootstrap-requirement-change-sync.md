# ADR-AIOPS-001: 운영체계 구축 중 요구사항 변경 동기화 강제

## Status

**Accepted** (2026-03-07)

## Context

운영체계 구축은 보통 문서 구조 변경, 규칙 정립, 템플릿 확정이 동시에 진행된다.
이 과정에서 요구사항이 자주 바뀌면 다음 문제가 반복된다.

- 작업계획(WP)과 실제 산출물(문서/폴더)이 어긋남
- 상태 전이만 진행되고 근거 문서가 늦게 갱신됨
- 결정(ADR) 누락으로 재작업이 발생함

특히 이번 운영체계 구축은 "구축 이후 적용"이 아니라 "구축 과정 자체부터 WP/ADR로 운영"하는 것이 목표이므로,
요구사항 변경 시 동기화 강제가 없으면 목표를 달성할 수 없다.

또한 프롬프트 기반 소프트 훅만으로는 다음 리스크가 남는다.

- 컨텍스트가 꼬이면 규칙 누락/체리피킹 가능
- 도구별 실행 레이어가 다르면 강제 지점이 불명확해짐
- 최종 산출물 수준에서 준수 여부를 재검증하지 못함

## Decision

### 1. 강제 체계를 2개 트랙으로 분리한다

운영체계 강제는 다음 두 계층을 동시에 운영한다.

- 트랙1: 작업 과정 강제 (AI tool별 훅/게이트)
- 트랙2: 최종 산출물 강제 (Git hook 기반 준수 검토)

### 2. 트랙1: 작업 과정 강제 (AI tool별)

도구별 적용 원칙을 다음처럼 고정한다.

1. Codex(JetBrains AI Assistant): AI Assistant 규칙 추가로 절차 준수 강제
2. Claude Code: `PreToolUse` 훅으로 필수 갱신 전 차단/질의
3. OMC: `quality gates`, `Ultra QA` 검증 루프를 경유해 품질 게이트 통과 후 진행

### 3. 트랙2: 최종 산출물 강제 (Git hook)

커밋/푸시 직전에 Git hook으로 산출물을 점검한다.

1. 운영체계 준수 흔적(WP/index/CHANGELOG/ADR) 확인
2. 준수 의심 항목 발견 시 차단 또는 리뷰 요구
3. 통과하지 못하면 최종 반영 단계로 진행 불가

### 4. 훅은 고정 산출물이 아니라 지속 업데이트 대상이다

트랙1/트랙2 훅은 운영체계 고도화 과정에서 계속 업데이트한다.

1. 신규 프로세스/규칙이 추가되면 훅 규칙을 같은 릴리즈 주기에 갱신
2. 오탐/미탐 또는 우회 사례가 확인되면 훅 규칙 즉시 보정
3. 정기 점검 주기(예: 주간/마일스톤)마다 훅 유효성 리뷰 수행

### 5. 요구사항 변경 동기화 항목을 강제한다

요구사항이 추가/수정/삭제되면 같은 세션(또는 같은 작업 단위)에서 아래를 모두 갱신한다.

1. 부모 WP: Scope, Done Criteria, Process Plan, Timeline
2. 영향받는 자식 WP: 상태/범위/후속 작업
3. `work-packets/index.md`: active/on-hold/우선순위
4. `CHANGELOG.md`: 변경 이력 한 줄 요약
5. ADR: 정책/원칙 변경 시 신규 ADR 또는 기존 ADR 보강

### 6. 하드 게이트를 적용한다

위 동기화가 누락되면 다음 프로세스로 상태 전이를 금지한다.

- 예: `IN_REFINEMENT -> IN_DEVELOPMENT` 전이 금지
- 예: `IN_DEVELOPMENT -> IN_REVIEW` 전이 금지

### 7. 완료 상태도 되돌릴 수 있다

요구사항 변경이 기존 완료 조건을 깨면, 이미 `DONE`인 WP도 `IN_REFINEMENT` 또는 `IN_DEVELOPMENT`로 되돌린다.
완료 상태는 최종 고정값이 아니라 조건 충족의 결과로 본다.

## Consequences

### 긍정적
- 작업 중 훅 + 최종 결과물 훅의 이중 방어가 생긴다.
- 훅 규칙이 누적 학습되며 운영체계 성숙도에 맞춰 진화한다.
- 운영체계 구축 중 문서/상태/결정의 정합성이 유지된다.
- 계획이 실제 실행을 따라가며, 요구사항 변경 이력이 추적 가능해진다.
- "나중에 정리" 패턴이 줄어들어 재작업 비용이 감소한다.

### 부정적
- 초기에는 갱신 오버헤드가 증가한다.
- 도구별 훅 설정/유지보수 비용이 추가된다.
- 작은 변경에도 기록이 필요해 체감 속도가 낮아질 수 있다.
