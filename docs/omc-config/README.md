# OMC Config Templates

This directory contains optional policy templates for repositories that also use OMC.

## Included Files

- `AI-OPS-POLICY.template.md` — portable policy text that can be copied into `.omc/AI-OPS-POLICY.md` (legacy filename, reusable for ReproGate)

## Apply

If a target repository wants a local OMC policy file, copy the template into `.omc/AI-OPS-POLICY.md` and adjust any project-specific references there rather than in the shared framework docs.
