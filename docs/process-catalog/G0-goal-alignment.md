# G0. Goal Alignment Process

> Status: Active

## Purpose

Confirm that the work still aligns with the higher-level operating goals before execution starts.
`G0` is the prerequisite process for all other processes.

## Use When

- starting a new AI session,
- starting work on a new packet or task,
- changing from one process to another,
- or pausing long work for a midpoint check.

## Inputs

- framework constitution,
- operating model,
- active project plan or control board,
- active work packet or equivalent task record.

## Steps

1. Identify which North Star goals the work supports.
2. Confirm the current phase or program position.
3. Check for goal drift.
4. Check for missing prerequisites or skipped packets.
5. Confirm the active packet or task status.
6. Decide whether work is clear, blocked, or needs clarification.

## Required Deliverables

- G0 result table
- proceed / clarify / stop decision

## Exit Criteria

- higher-level alignment is confirmed,
- the active task is identified,
- and the next action is explicit.

- [ ] constitution reviewed
- [ ] operating model reviewed
